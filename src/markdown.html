STATS BAYES REGR Extension Command

Calculate Bayesian Regression

STATS BAYES REGR DEPENDENT = dependent variable*
INDEP = independent variables*
MODELS = ALLMODELS** or STEPDOWN or STEPUP or SINGLE

/OPTIONS COMPARISON=model number
POSTERIORINDEX = model number
ITERATIONS = number
PLOTMODELS = NO** or YES
PRIORSCALE = MEDIUM** or WIDE or ULTRAWIDE
MAXMODELS = number

/SAVE WORKSPACE = CLEAR or RETAIN
MODELFILE = "file specification"

* Required
** Default

STATS BAYES REGR /HELP prints this information and does nothing else.

STATS BAYES REGR DEPENDENT=y
INDEP=x1 x2 x3
/OPTIONS COMPARISON=7.

Split files and weights are not honored by this procedure. Cases with missing values are omitted.

The procedure calculates the Bayes factors and the posterior probabilities for a set of models using various combinations of the independent variables as specified in the MODELS keyword. The distributions for a selected model can also be displayed. The models are numbered for reference in the COMPARISON and POSTERIORINDEX keywords, so it is useful to first run the procedure without using them in order to see how the models are numbered.

DEPENDENT and INDEP specify the dependent and independent variable names. All variables must have a scale (continuous) measurement level. measurement level. A constant term is automatically included.

MODELS specifies the combinations of independent variables to be used.

  • ALLMODELS produces results for all combinations of the variables.
  • STEPDOWN produces all the models where exactly one independent variable is omitted.
  • STEPUP produces all the models that include a single independent variable.
  • SINGLE produces a single model containing all the independent variables. In this case, the COMPARISON, POSTERIORINDEX, and PLOTBAYESF settings do not apply, and the posterior distribution is always produced.

OPTIONS

COMPARISON specifies the based model used to calculate the Bayes factors. By default, that is an intercept-only model. Specify the model number to use it as the base instead.

POSTERIORINDEX specifies that the posterior distributions of the coefficients be specified for the selected model number. If this keyword is not specified, the posterior distributions are not produced except in the single variable case.

ITERATIONS specifies the number of iterations (samples) to use for the posterior. The default value is 1000.

PRIORSCALE specifies the prior scale on all standardized slopes

PLOTMODELS produces a plot of the Bayes factors. With more than a few models, however, the plot becomes unreadable.

MAXMODELS specifies the maximum number of models to estimate. If ALLMODELS are estimated, the number to estimate can quickly become very large and take a long time to compute. The default value is 10,000.

SAVE

WORKSPACE specifies whether the workspace should be cleared or retained after the procedure completes. By default, it is cleared.

MODELFILE specifies a file name to which the model information will be written as an R workspace for later use.
