Bayesian Regression

The procedure calculates the Bayes factors and the posterior probabilities for a set of models using various combinations of the independent variables as specified in the Models box. The posterior distributions for a single selected model can also be displayed. The models are numbered for reference in the Model Comparisons and Posterior Distribution fields, so it is useful to first run this procedure without using these numbers in order to see how the models are numbered.

The Bayes factor is the relative evidence in the data for one model over another. See the reference below for an introductory discussion.

Split files and the Statistics case weights are not supported by this procedure. Cases with missing values will be omitted from the analysis.

Dependent Specify the dependent variable.

Independent Specify the independent variables. Only continuous variables can be used. You can dichotomize categorical variables and use them here as long as the measurement levels are set to scale.

Models This setting determines which models will be estimated from the set of independent variables.

  • All produces results for all combinations of the variables.
  • Single Deletion produces all the models where exactly one independent variable is omitted.
  • Single Variable produces all the models that include a single independent variable.
  • Single Model with All Variables produces a single model containing all the independent variables. In this case, the Model Comparisons, Posterior Model Number, and Plot models Bayes Factors settings on the Options subdialog do not apply, and the posterior distribution is always produced.

    Options

    Model Comparisons By default, the Bayes factors are computed relative to an intercept-only model. You can change the basis by specifying a model number. You can see how the models are numbered by running the specification once without using this option.

    Posterior Distribution You can display the posterior distribution of the model coefficients for a single model by specifying the model number.

    Number of samples Specify the number of iterations (samples) to be drawn in calculating the posterior distribution.

    Plot model Bayes factors Check this box to display a graph of the Bayes factors. If the number of models is more than about ten, however, the chart will be very difficult to read.

    Maximum Number of Models Specify a limit on the number of models to be estimated.

    Prior Scale Choose the scale for the prior on all the standardized slopes. The scale parameter controls the expected true standardized effect size. For a particular scale 50% of the true effect sizes are within the interval. For "medium", this is (-.7071, .7071). Wide is (-1, 1), and ultrawide is (-1.414, 1.414).

    Additional Features

    This dialog generates syntax for the STATS BAYES REGR extension command. To display syntax help for this command, run the following syntax:

    STATS BAYES REGR /HELP.

    In Statistics version 23 or later, you can place the cursor within the command in the Syntax Editor and press F1.

    Using syntax you can save the R workspace to a file or keep it in memory for use by an R program run within Statistics.
